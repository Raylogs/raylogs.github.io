Prox VC install guide

Proximity voice chat + fabric install guide

Step 1:

Download Fabric, Fabric API and Plasmovoice.

Step 2:

Open fabric-installer.exe and install fabric.
(make sure you have launched vanilla mc 1.20.1 atleast once)

Step 3:

Press the windows key, and type `%appdata%` and then press enter.
Then go to the .minecraft folder and inside create a folder named `mods`.

Step 4:

Put Fabric-api.jar and Plasmovoice-fabric.jar in the folder.

Step 5:

Launch minecraft using the fabric profile (automatically selected) and join `raylog.lol`.

Step 6:

Type `/vrc` in chat and then press V. (if pressing V doesn't work check your keybinds)

Step 7:

Adjust settings to whatever as long as you can hear others and can be heard.
Also make sure to turn on voice activation.
